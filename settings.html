General Settings

User interface
Adds an icon to username fields for filling credentials with a single mouse click.
Adds an icon to password fields for generating a new password.
Passwords are generated by KeePassXC using your password generation profile.
Adds an icon to input fields that are recognized as OTP/2FA.
Show notifications for errors and when user interaction is required.
Keyboard shortcuts

Fill Username and Password : Alt+Shift+U
Fill Password Only : Alt+Shift+I
Fill TOTP : Alt+Shift+O
Show Password Generator : Alt+Shift+G

Filling credentials
KeePassXC-Browser will immediately retrieve credentials when a tab is activated.
Show a dropdown list containing available credentials for all username fields on a page.
When Relevant credential first option is selected, the credential for previously filled login is set as first in Autocomplete Menu's listing.
When Relevant credential first option is selected, the TOTP for previously filled login is set as first in Autocomplete Menu's listing.
Automatically submits selected credentials. This is disabled for single entry auto-fill.
Let KeePassXC-Browser automatically fill in credentials if it receives only a single entry.
Warning! Using auto-fill is not safe. Use at your own risk. KeePassXC-Browser automatically tries to detect login fields. However, they may be detected incorrectly, possibly filling sensitive data to unsafe input fields.
If credentials are found for a page and the login-type is an HTTP Basic Auth request, KeePassXC-Browser tries to login automatically with the first given credentials. When multiple credentials are present, the extension popup can be used to choose the correct one. Even when this feature is enabled, it is still possible to enter HTTP Basic Auth login manually.

Saving credentials
When enabled, the popup shows a group listing.
Separate the group with slashes, for example: Group/ChildGroup.
When saving new credentials, save only the domain instead of full URL.
Favicon will be automatically downloaded for the new credential.
How many redirects are allowed until the banner disappears. Default value is 3. Maximum is infinite.

Site preferences

Sites on this page have special handling methods associated with them.

  • To ignore new/modified credentials on a specific site, add them below or when using the credential banner on the page select Don't ask again for this site .
  • If a site is fully ignored ( Disable all features is selected), then the plugin will do nothing when visiting that site.
  • Username-only detection allows KeePassXC-Browser to fill in login details on websites with separate pages for username and password.
  • Improved Input Field Detection allows more detailed dynamic input field detection. However, it might affect the page performance. Use with caution.
The URL must start with either https:// , http:// , file:// , or ftp:// and must be at least 5 characters long.
